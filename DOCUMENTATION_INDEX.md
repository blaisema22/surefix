## üéì Key Concepts

**Hooks**: Functions that handle API calls and state
**API Client**: Central place for all HTTP requests
**Token**: JWT used to authenticate requests
**Loading State**: Says if data is fetching
**Error State**: Message if something fails
**localStorage**: Where token is stored
